BMI Calculator

Overview: A user-friendly tool to calculate Body Mass Index (BMI) based on user-provided weight and height.

Functionality: Users can input their weight (kg) and height (cm) to compute their BMI, categorized as underweight, normal, overweight, or obese.

Technologies Used: HTML, CSS, JavaScript.

To-Do List

Overview: A task management application that allows users to organize and prioritize tasks effectively.

Functionality: Users can add, edit, and delete tasks, with tasks persisting across sessions using local storage.

Technologies Used: HTML, CSS, JavaScript.

Unlimited Color Changer

Overview: An interactive web application that allows users to change the background color of the page dynamically.

Functionality: Users can select any color from a color picker, instantly changing the background color.

Technologies Used: HTML, CSS, JavaScript.
